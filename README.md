# 学生成绩管理
## 需求分析

​        学生管理系统是一个教育单位不可缺少的部分，它对于学校的管理者来说是重要的，所以学生成绩管理系统应该能够为用户提供需要的信息的查询手段，是为了方便学校管理学生信息，运用学生管理系统对学生的信息进行管理，不仅可以方便学校对于所有学生的信息进行系统的管理，而且便于教师对学生成绩进行查询和修改。
